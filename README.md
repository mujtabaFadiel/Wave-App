# Social Media App

## Description
A simple social media app built with:
- NestJS (Backend)
- React Native (Mobile)

Users can:
- Register & Login (JWT Auth)
- Create tweets
- Like & Comment
- Follow users
- View feed (following users only)

---

##  Features
- Authentication with JWT
- Tweets CRUD
- Like system
- Comment system
- Follow system
- Feed system

---

##  What I learned
- Building REST APIs with NestJS
- JWT Authentication
- Database relations (User, Tweet, Like, Comment, Follow)
- Handling real-world logic (feed, follow, likes)
